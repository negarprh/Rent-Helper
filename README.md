# RentHelper

RentHelper is an iOS application that helps users find rental listings and save favorites for later viewing.

## Tech Stack
- SwiftUI
- Firebase Authentication
- Firebase Firestore
- CoreData
- MapKit
- Stripe (planned)

## Current Features (30% Deliverable)
- User authentication (signup/login/logout)
- Fetch rental listings from Firestore
- View listing details
- Save listings to favorites (CoreData)
- Favorites persist after app restart

## How to Run
1. Clone the repo
2. Open `RentHelper.xcodeproj`
3. Run on iOS Simulator
